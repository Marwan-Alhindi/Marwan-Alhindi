# Hi, I’m Marwan

I build agents.

I started my undergraduate in data science and became interested in natural language processing. That's where I ran into the problem that defines the field: natural language is ambiguous and context-dependent, and statistics can only approximate it — never guarantee it. That pushed me toward a master's in computer science, where I studied the opposite side: formal language systems.The one thing I took from both sides: natural language and formal language systems are each essential. Today I work and study as an AI Engineer, and the role sits right where those two worlds meet.

At the heart of AI Engineering is knowing where to draw the line between formal systems and natural language models, and building agents. Narrow, well-defined problems belong to code where its deterministic, efficient, verifiable. Open-ended problems with unbounded input belong to the model. The engineer’s job is deciding which is which.
