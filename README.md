# Hi, I’m Marwan

That pushed me toward a master's in computer science, where I studied the opposite side: formal language systems.The one thing I took from both sides: natural language and formal language systems are each essential. Today I work and study as an AI Engineer, and the role sits right where those two worlds meet.
